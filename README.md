GShot
=====

GShot is a fork of the now abandoned GNOME Screenshot

It is a small utility that takes a screenshot of the whole
desktop; the currently focused window; or an area of the screen.

The standard GNOME desktop session does not use GNOME Screenshot any more.

### Dependencies

 - GLib 2.36
 - GTK+ 3.12
 - Wayland
 - X11
